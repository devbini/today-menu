# 🥘 구내식당 메뉴 미리보기 웹 서비스

## 👏 소개
>  
> 구내식당 미리보기 서비스는 사옥 내 **구내식당의 메뉴를 미리 확인하지 못하는 불편함을 해소하기 위해** 개발된 서비스입니다.  
> 이 서비스는 **식당 측에서 간편하게 메뉴를 관리할 수 있도록**,  
> 간단히 사진 한 장과 사이드 메뉴 이름만 저장할 수 있게 설계되었습니다.  

## 🚀 접속 주소
- **Production URL : https://woorung.kr/**
